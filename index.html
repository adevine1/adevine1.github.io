Get Five!

Get Five! is a clone of a popular 5-dice game (i.e., Yahtzee) that I wrote in HTML/CSS/JS/JQuery. It is based on a hand-held device that I have played to death over the years. I cloned it a number of years ago for Windows (in Delphi), and I also wanted to see if I could do it for the web. Get Five! uses local storage to persist high-score data between sessions.
