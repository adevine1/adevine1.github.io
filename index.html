Get Five!
Get Five! is a clone of a popular 5-dice game (i.e., Yahtzee) that I wrote in HTML/CSS/JS/JQuery. It is based on a hand-held device that I played quite a bit over the years. I cloned it a number of years ago for Windows (in Delphi), and this is a similar sort of project in JavaScript. Get Five! uses local storage to persist high-score data and user settings between sessions.
